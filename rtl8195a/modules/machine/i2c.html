Class I2C¶

class I2C(unit=0, mode=I2C.MASTER, baudrate=100000, pins=("PD_5", "PD_4"))¶

Constructor to create a machine.I2C object, MASTER mode only

參數:
  • unit (int) – unit
  • mode (const) – I2C mode, MASTER only.
  • baudrate (int) – operating Hz
  • pins (tuple) – alternative pins (SDL, SDA)
返回:

machine.I2C object

unit SCL SDA
0 PD_5 PD_4
1 PD_6 PD_7
1 PC_5 PC_4
2 PE_0 PE_1
3 PB_2 PB_3
3 PE_2 PE_3
3 PE_4 PE_5 

>>> from machine import I2C
>>> device = I2C(1, I2C.MASTER, baudrate=200000, pins=("PD_6", "PC_4"))
>>> print(device)

Methods¶

I2C.reset()¶

Reset I2C core.

返回:None
I2C.recv(nbytes, addr, stop=True)¶

Read value from device with n bytes, user can handle the stop bit by assigning stop=False/True

參數:
  • nbytes (int) – to read n bytes
  • addr (hex) – slave device’s address
  • stop (bool) – to generate the stop bit or not
返回:

received data
返回類別:

bytearray

 
>>> from machine import I2C
>>> device = I2C(0, I2C.MASTER)
>>> result = device.recv(4, 0x50)
>>> print(result)
>>> print(result[0])
>>> print(result[1])
>>> print(result[2])
>>> print(result[3])
I2C.send(buf, addr, stop=True)¶

Send buffer to slave device, user can handle the stop bit by assigning stop=False/True

參數:
  • buf (btearray) – content
  • addr (hex) – slave device’s address
  • stop (bool) – to generate the stop bit or not
返回:

bytes write

 
>>> from machine import I2C
>>> device = I2C(0, I2C.MASTER)
>>> device.send(0xAA, 0x50, stop=True)
1
>>> byte_list = [0x01, 0x02, 0x03, 0x04, 0x05]
>>> buffer = bytearray(byte_list)
>>> device.send(buffer, 0x50, stop=False)
5
I2C.mem_read(nbytes, addr, mem_addr, addr_size=8)¶

Read memory value inside the device with n bytes. Some I2C device are 16 bits addressing, so user can assign addr_size=16 for that.

參數:
  • nbytes (int) – to read n bytes
  • addr (hex) – slave device’s address
  • mem_addr (hex) – memory offset inside the device
  • addr_size (int) – 8 bits or 16 bits addressing
返回:

received data
返回類別:

bytearray
