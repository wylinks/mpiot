wireless – a module to access wireless functions¶

class WLAN¶

Constructor¶

class WLAN(mode=WLAN.AP, ssid="mpiot-ap", auth=(WLAN.WPA2_AES_PSK, "1234567890"), channel=6)¶

Constructor to create a WLAN object, support STA (station), AP and STA_AP cocurrency mode.

參數:
  • mode (const) – wlan mode, available modes: wifi mode
  • ssid (str) – your ssid in WLAN.AP and WLAN.STA_AP modes.
  • auth (tuple) – your security type and password in WLAN.AP and WLAN.STA_AP modes. available types: security type
  • channel (int) – wifi channel in WLAN.AP and WLAN.STA_AP modes.
Raises:

ValueError –

  1. if channel is not at the range of 1 ~ 11.
  2. if ssid length is larger than 32 characters.
  3. if password length is larger than 32 characters.
返回:

WLAN object

WLAN.STA mode example, you can connect to an AP

>>> from wireless import WLAN
>>> wifi = WLAN(mode=WLAN.STA)
>>> try:
...     wifi.connect("TARGET-SSID", auth=(WLAN.WPA2_AES_PSK, "TARGET-PASSWORD"))
... except OSError:
...     print("connection failed")

WLAN.AP mode example, you are in AP mode only.

>>> from wireless import WLAN
>>> wifi = WLAN(mode=WLAN.AP, ssid="your-ap-ssid", auth=(WLAN.WPA_AES_PSK, "your-ap-password"), channel=3)

WLAN.STA_AP mode example, you are in AP mode (use your PC to scan the WiFi) and also you can connect to a nearby AP at the same time.

>>> from wireless import WLAN
>>> wifi = WLAN(mode=WLAN.AP, ssid="your-ap-ssid", auth=(WLAN.WPA_AES_PSK, "your-ap-password"), channel=3)
>>> try:
...     wifi.connect("TARGET-SSID", auth=(WLAN.WPA2_AES_PSK, "TARGET-PASSWORD"))
... except OSError:
...     print("connection failed")

Methods¶

WLAN.scan()¶

To scan all available networks nearby.

(workable in mode WLAN.STA, WLAN.AP, WLAN.STA_AP, WLAN.P2P and WLAN.PROMISC)

返回:lists of tuple 
>>> from wireless import WLAN
>>> wifi = WLAN(mode=WLAN.STA)
>>> for i in range(10):
...     ap_list = wifi.scan()
...     for ap in ap_list:
...           print("ssid = %s, bssid = %s, rssi = %d, channel = %d" % (ap.ssid, ap.bssid, ap.rssi, ap.channel))
WLAN.connect(ssid, auth)¶

To connect to a WiFi AP.

參數:
  • ssid (str) – your target AP ssid
  • auth (tuple) – your target security type and password
Raises:

OSError –

  1. wrong ssid, security type or password
  2. you are in WLAN.AP mode
返回:

None

 
>>> from wireless import WLAN
>>> wifi = WLAN(mode=WLAN.STA)
>>> try:
...     wifi.connect("YOUR_SSID", auth=(WLAN.WPA2_AES_PSK, "YOUR_PASSWORD"))
... except OSError as e:
...     print("access point auth failed")
...     print(e)
