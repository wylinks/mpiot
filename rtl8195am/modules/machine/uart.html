Class UART¶

class UART(unit=0, baudrate=9600, bits=8, stop=1, parity=UART.ParityNone, timeout=0, timeout_char=0, pins=("PA_7", "PA_6"))¶

Constructor to create a machine.UART object.

參數:
  • unit (int) – unit
  • baudrate (int) – baudrate
  • bits (int) – bits
  • stop (int) – numbers of stop bit
  • parity (const) – parity
  • timeout (int) – timeout
  • timeout_char (char) – timeout_char
  • pins (tuple) – alternative pins (TX, RX)
返回:

machine.UART object

unit TX RX
0 PA_7 PA_6
0 PC_3 PC_0
0 PE_0 PE_3
2 PA_4 PA_0
2 PD_7 PD_4 

>>> from machine import UART
>>> device = UART(2, baudrate=115200, pins=("PA_4", "PD_4"))
>>> print(device)
