Class DAC¶

class DAC(unit=0, resolution=10)¶

Constructor to create machine.DAC object. Only unit 0 is available.

unit DAC
0 DA_0 
>>> from machine import DAC
>>> dac = DAC(resolution=12)
>>> for i in range(pow(2, 12)):
...     dac.write(i)

Methods¶

DAC.write(value)¶

Write digital value to DAC.

參數:value (int) – digital value depends on your resolution.
返回:None

註解

DAC voltage range is 0 ~ 3.3V.
