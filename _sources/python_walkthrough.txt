Python Walkthrough
======================

Why use Python ?

    一般來說, 要使用開發版進行應用開發, 需要透過以下幾個步驟

    1. 撰寫程式
    2. 編譯
    3. 程式燒錄
    4. 監看運行狀況

    但是進行開發過程中, 往往會出現相當多無法預期的狀況, 例如程式碼邏輯不清楚, 程式流程設計錯誤, 甚至是變數宣告錯誤, 而造成程式運行出問題, 為了修改一個小地方, 而必須要重新進行以上4個步驟.
    往往沒有經驗的人, 對於要設計一個稍微複雜的物聯網應用, 會需要反反覆覆的進行除臭蟲, 程式編譯, 程式燒錄, 監看運行結果. 
    這種開發方式往往是相當沒有效率的
